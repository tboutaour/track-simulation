Commands¶

analysis_main.analysis_main(file_directory, north_component, south_component, east_component, west_component)¶

Main function to deploy pipeline that analyzes GPX tracks in directory

Parameters

  • file_directory – directory path where files are stored.

  • north_component – North component of limited area.

  • south_component – South component of limited area.

  • east_component – East component of limited area.

  • west_component – West component of limited area.

simulation_main.simulation_main(origin_node, distance, data, quantity)¶

Main function to deploy pipeline that simulates track based on previous analysis.

Parameters

  • origin_node – Node to start simulation.

  • distance – Distance to start simulation.

  • data – Data to import from database. (Graph_Analysis_mm-dd-YYYY)

  • quantity – Number of tracks to generate.
